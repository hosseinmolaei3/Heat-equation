سلام و عرض خسته نباشید
این این سند با استفاده از زبان جاوااسکریپت نوسته شده است یک جدول در میانه صفحه وجود دارد که شما میتوانید مقادیر را درآن وارد کنید از u0 وdt وdx وalpha و n که تعداد جملات است مقادیر را در جدول پایین تر نشان شما میدهد
در پایین آن نمودارش را رسم میکند امیدوارم رابط کاربری ساده ای طراحی کرده باشم
Variable Explanation Value
u0 Initial temperature distribution
alpha Thermal diffusivity of the material
dt Time step size
dx Spatial step size
n Number of time steps to simulate
